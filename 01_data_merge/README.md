# 数据合并模块

## 功能说明
此模块负责将多个单细胞RNA测序样本的原始数据进行合并，并进行初步的质量评估。

## 主要功能
- 读取10X Genomics格式的过滤矩阵
- 合并多个样本数据
- 计算线粒体和叶绿体基因表达百分比
- 生成质控统计图表（小提琴图和散点图）
- 保存合并后的Seurat对象

## 文件列表
- `00mergeTask.ipynb` - 数据合并主脚本

## 输入数据
- 10X Genomics `filter_matrix` 目录
- 基因注释文件（geneinfo.txt）

## 输出结果
- 合并后的RDS文件（`*_merge_raw.rds`）
- 质控统计表格（`*_qc_stat_raw.tsv`）
- 质控可视化图（VlnPlot和Scatter图）

## 使用说明
1. 设置参数：样本路径、基因注释文件路径、输出路径
2. 运行notebook中的所有代码块
3. 检查输出的质控图，为下一步QC参数设置提供依据
